# Mockups & Visual References

Visual references for Siren's UI design. Each mockup is tied to a specific feature drop.

## Naming Convention
`{feature}_{view}_{version}.png` — e.g., `inspector_page_selector_v1.png`

## Index

### v1.0 — FastPages

| File | Description |
|------|-------------|
| `mockup_v1.png` | Original wireframe showing the full FastPages flow: empty state → new project → project detail with pages → page detail with selectors → inspector with element mapping. Hand-drawn on dark background. |
| `project_dashboard_v1.png` | Implemented projects dashboard. Shows sidebar with navigation, project cards with name/URL/description, "+ New Project" button. Dark theme. |
| `inspector_page_v1.png` | Inspector in idle state. Target website (demoqa.com Practice Form) loaded via proxy in iframe. Right sidebar shows "Selector Options" with instructions to click "Get Selector" first. Top bar shows selector name and URL. |
| `inspector_page_selector_v1.png` | Inspector in active state. User has clicked on the "First Name" input. Right sidebar shows extracted selector options: id (`#firstName`), placeholder, class, css path, and tag. Element preview thumbnail at top. "Click an element..." button active in top-right. |
